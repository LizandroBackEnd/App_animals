# Uso del proyecto 
Clona o descarga el proyecto 
```bash
git clone https://github.com/LizandroBackEnd/practice_dog.git
```  
 
Instala las dependencias 
```bash
npm install
```   
 
Corre la aplicacion 
```bash
npm start
```   
Escanea el codigo QR con la app de EXPO GO en tu movil o emulador.
